TextNode

class TextNode extends DirectEntryWindowNode (View source)

TextNode is displayed as a ModalForm that renders the translated message specified in the content attribute.

. The child nodes <button1> and <button2> can be specified to change the messages and behaviour. By default, they are "Back" and "Quit".

Note that there is no button to close a ModalForm, so the player must either click button1 or button2, unless they quit the game.
